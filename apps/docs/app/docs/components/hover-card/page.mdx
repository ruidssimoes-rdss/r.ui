# HoverCard

A floating preview card that appears when hovering (web) or long-pressing (mobile) a trigger element.

## Installation

```bash
npx r-ui add hover-card
```

## Usage

```tsx
import {
  HoverCard,
  HoverCardTrigger,
  HoverCardContent,
} from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <HoverCard>
      <HoverCardTrigger>
        <Text style={{ color: '#3b82f6' }}>@username</Text>
      </HoverCardTrigger>
      <HoverCardContent>
        <Avatar src="..." />
        <Text>User Name</Text>
        <Text>Bio goes here...</Text>
      </HoverCardContent>
    </HoverCard>
  )
}
```

## Positions

Control where the hover card appears relative to the trigger.

```tsx
// Top
<HoverCard side="top">
  <HoverCardTrigger>
    <Text>Hover me</Text>
  </HoverCardTrigger>
  <HoverCardContent>
    Content appears above
  </HoverCardContent>
</HoverCard>

// Bottom (default)
<HoverCard side="bottom">
  ...
</HoverCard>

// Left
<HoverCard side="left">
  ...
</HoverCard>

// Right
<HoverCard side="right">
  ...
</HoverCard>
```

## Alignment

Control how the card aligns with the trigger.

```tsx
// Start alignment
<HoverCard align="start">
  ...
</HoverCard>

// Center alignment (default)
<HoverCard align="center">
  ...
</HoverCard>

// End alignment
<HoverCard align="end">
  ...
</HoverCard>
```

## Custom Delays

Customize the open and close delays.

```tsx
// Instant open/close
<HoverCard openDelay={0} closeDelay={0}>
  ...
</HoverCard>

// Fast open (500ms)
<HoverCard openDelay={500} closeDelay={200}>
  ...
</HoverCard>

// Slow open (1 second)
<HoverCard openDelay={1000} closeDelay={500}>
  ...
</HoverCard>
```

## Controlled

Control the hover card state programmatically.

```tsx
import { useState } from 'react'
import {
  HoverCard,
  HoverCardTrigger,
  HoverCardContent,
  Button,
} from '@r-ui/react-native'

function ControlledHoverCard() {
  const [open, setOpen] = useState(false)

  return (
    <>
      <Button onPress={() => setOpen(!open)}>
        Toggle HoverCard
      </Button>
      <HoverCard open={open} onOpenChange={setOpen}>
        <HoverCardTrigger>
          <Text>Controlled Trigger</Text>
        </HoverCardTrigger>
        <HoverCardContent>
          <Text>Controlled content</Text>
        </HoverCardContent>
      </HoverCard>
    </>
  )
}
```

## Mobile Behavior

On mobile devices where hover is not available, the HoverCard activates on long-press instead. Users can long-press the trigger to show the card, which will close when they release or tap elsewhere.

## Props

### HoverCard

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `openDelay` | `number` | `700` | Delay before opening (ms) |
| `closeDelay` | `number` | `300` | Delay before closing (ms) |
| `side` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Side to display content |
| `align` | `'start' \| 'center' \| 'end'` | `'center'` | Alignment relative to trigger |
| `sideOffset` | `number` | `8` | Offset from trigger |

### HoverCardTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger element |
| `style` | `ViewStyle` | - | Additional styles |

### HoverCardContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content |
| `style` | `ViewStyle` | - | Additional styles |
