# Badge

Small status indicator for labeling and categorization.

## Installation

```bash
npx r-ui add badge
```

## Usage

```tsx
import { Badge } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Badge>New</Badge>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| 'outline'` | `'default'` | Visual style variant |
| `size` | `'sm' \| 'md'` | `'md'` | Badge size |
| `style` | `ViewStyle` | - | Additional container styles |
| `textStyle` | `TextStyle` | - | Additional text styles |
