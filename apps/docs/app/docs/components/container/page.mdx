# Container

Centered max-width wrapper for constraining content width.

## Installation

```bash
npx r-ui add container
```

## Usage

```tsx
import { Container } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Container size="md" padding="md">
      <Text>Contained content</Text>
    </Container>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'` | `'lg'` | Maximum width size |
| `padding` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Horizontal padding |
| `centered` | `boolean` | `true` | Center the container horizontally |
| `style` | `ViewStyle` | - | Additional styles |
