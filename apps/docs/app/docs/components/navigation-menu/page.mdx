# NavigationMenu

Site navigation component with dropdown content panels for main navigation.

## Installation

```bash
npx r-ui add navigation-menu
```

## Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
} from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Products</NavigationMenuTrigger>
          <NavigationMenuContent>
            <NavigationMenuLink href="/products/new">New Arrivals</NavigationMenuLink>
            <NavigationMenuLink href="/products/sale">Sale</NavigationMenuLink>
          </NavigationMenuContent>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  )
}
```

## Props

### NavigationMenu

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction |
| `delayDuration` | `number` | `200` | Delay before opening (ms) |
| `style` | `ViewStyle` | - | Additional styles |

### NavigationMenuList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Menu items |
| `style` | `ViewStyle` | - | Additional styles |

### NavigationMenuItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger and content |
| `style` | `ViewStyle` | - | Additional styles |

### NavigationMenuTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger text/content |
| `style` | `ViewStyle` | - | Additional styles |

### NavigationMenuContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Dropdown content |
| `style` | `ViewStyle` | - | Additional styles |

### NavigationMenuLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | Link URL |
| `onPress` | `() => void` | - | Custom press handler |
| `active` | `boolean` | `false` | Show as active |
| `style` | `ViewStyle` | - | Additional styles |
