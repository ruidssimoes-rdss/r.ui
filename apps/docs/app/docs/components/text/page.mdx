# Text

Typography component for rendering text with consistent styling across your app.

## Installation

```bash
npx r-ui add text
```

## Usage

```tsx
import { Text } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Text variant="h1">Welcome to r.ui</Text>
  )
}
```

## Convenience Components

The library also exports convenience components for common use cases:

```tsx
import { Heading, Code, Label } from '@r-ui/react-native'

// Heading with level prop
<Heading level={1}>Main Title</Heading>
<Heading level={2}>Section Title</Heading>

// Code text
<Code>npm install @r-ui/react-native</Code>

// Label text
<Label>Form Field Label</Label>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'body' \| 'bodySmall' \| 'label' \| 'labelSmall' \| 'code'` | `'body'` | Typography variant |
| `color` | `'primary' \| 'secondary' \| 'muted' \| 'inverse' \| 'inherit'` | `'primary'` | Text color |
| `weight` | `'normal' \| 'medium' \| 'semibold' \| 'bold'` | - | Font weight override |
| `align` | `'left' \| 'center' \| 'right'` | - | Text alignment |
| `style` | `TextStyle` | - | Additional styles |

## Typography Scale

| Variant | Font Size | Font Weight | Line Height |
|---------|-----------|-------------|-------------|
| `h1` | 36px | Bold | 1.25 |
| `h2` | 30px | Bold | 1.25 |
| `h3` | 24px | Semibold | 1.25 |
| `h4` | 20px | Semibold | 1.25 |
| `body` | 16px | Normal | 1.5 |
| `bodySmall` | 14px | Normal | 1.5 |
| `label` | 14px | Medium | 1.5 |
| `labelSmall` | 12px | Medium | 1.5 |
| `code` | 14px | Normal | 1.625 |
