# Pagination

Navigate through pages of content with previous/next controls.

## Installation

```bash
npx r-ui add pagination
```

## Usage

```tsx
import { Pagination } from '@r-ui/react-native'

export default function MyComponent() {
  const [page, setPage] = useState(1)

  return (
    <Pagination
      currentPage={page}
      totalPages={10}
      onPageChange={setPage}
    />
  )
}
```

## Props

### Pagination

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `currentPage` | `number` | - | Current page number (1-indexed, required) |
| `totalPages` | `number` | - | Total number of pages (required) |
| `onPageChange` | `(page: number) => void` | - | Callback when page changes (required) |
| `style` | `ViewStyle` | - | Additional styles |

### PaginationContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### PaginationItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### PaginationLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `page` | `number` | - | Page number (required) |
| `isActive` | `boolean` | `false` | Whether this is the current page |
| `onPress` | `(page: number) => void` | - | Click handler |
| `disabled` | `boolean` | `false` | Disable the link |
| `style` | `ViewStyle` | - | Additional styles |
| `textStyle` | `TextStyle` | - | Additional text styles |

### PaginationPrevious

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onPress` | `() => void` | - | Click handler |
| `disabled` | `boolean` | `false` | Disable the button |
| `label` | `string` | - | Optional text label |
| `style` | `ViewStyle` | - | Additional styles |

### PaginationNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onPress` | `() => void` | - | Click handler |
| `disabled` | `boolean` | `false` | Disable the button |
| `label` | `string` | - | Optional text label |
| `style` | `ViewStyle` | - | Additional styles |

### PaginationEllipsis

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |
