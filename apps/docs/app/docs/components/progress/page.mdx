# Progress

Progress bar for displaying completion status with animated fill.

## Installation

```bash
npx r-ui add progress
```

## Usage

```tsx
import { Progress } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Progress value={60} />
  )
}
```

## Dynamic Progress

The progress bar animates smoothly when the value changes.

```tsx
const [progress, setProgress] = useState(0)

useEffect(() => {
  const timer = setInterval(() => {
    setProgress((prev) => (prev >= 100 ? 0 : prev + 10))
  }, 500)
  return () => clearInterval(timer)
}, [])

<Progress value={progress} showValue />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Progress value (0-100) |
| `variant` | `'default' \| 'success' \| 'warning' \| 'error'` | `'default'` | Visual style variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Progress bar height |
| `showValue` | `boolean` | `false` | Show percentage label |
| `style` | `ViewStyle` | - | Additional container styles |
