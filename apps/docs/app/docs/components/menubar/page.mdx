# Menubar

Horizontal menu bar with dropdown menus, desktop-style File/Edit/View navigation.

## Installation

```bash
npx r-ui add menubar
```

## Usage

```tsx
import {
  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
} from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Menubar>
      <MenubarMenu>
        <MenubarTrigger>File</MenubarTrigger>
        <MenubarContent>
          <MenubarItem onPress={() => {}}>New File</MenubarItem>
          <MenubarItem onPress={() => {}}>Open</MenubarItem>
          <MenubarItem onPress={() => {}}>Save</MenubarItem>
        </MenubarContent>
      </MenubarMenu>
    </Menubar>
  )
}
```

## Props

### Menubar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction |
| `style` | `ViewStyle` | - | Additional styles |

### MenubarMenu

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Menu trigger and content |
| `style` | `ViewStyle` | - | Additional styles |

### MenubarTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger content |
| `style` | `ViewStyle` | - | Additional styles |

### MenubarContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Menu items |
| `align` | `'start' \| 'center' \| 'end'` | `'start'` | Content alignment |
| `style` | `ViewStyle` | - | Additional styles |

### MenubarItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onPress` | `() => void` | - | Press handler |
| `disabled` | `boolean` | `false` | Disable item |
| `shortcut` | `string` | - | Keyboard shortcut display |
| `style` | `ViewStyle` | - | Additional styles |

### MenubarCheckboxItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Checked state |
| `onCheckedChange` | `(checked: boolean) => void` | - | Change handler |
| `disabled` | `boolean` | `false` | Disable item |
| `shortcut` | `string` | - | Keyboard shortcut display |

### MenubarRadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Selected value |
| `onValueChange` | `(value: string) => void` | - | Change handler |

### MenubarRadioItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | Required | Item value |
| `disabled` | `boolean` | `false` | Disable item |
