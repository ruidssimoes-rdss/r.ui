# ActionSheet

iOS-style bottom sheet with action options. Slides up from the bottom for sharing, selecting options, or showing contextual actions.

## Installation

```bash
npx r-ui add action-sheet
```

## Usage

```tsx
import {
  ActionSheet,
  ActionSheetTrigger,
  ActionSheetContent,
  ActionSheetHeader,
  ActionSheetTitle,
  ActionSheetItem,
  ActionSheetSeparator,
  ActionSheetCancel,
} from '@r-ui/react-native'
import { Button } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <ActionSheet>
      <ActionSheetTrigger asChild>
        <Button>Share</Button>
      </ActionSheetTrigger>
      <ActionSheetContent>
        <ActionSheetHeader>
          <ActionSheetTitle>Share this post</ActionSheetTitle>
        </ActionSheetHeader>
        <ActionSheetItem onPress={() => share('twitter')}>
          Share on Twitter
        </ActionSheetItem>
        <ActionSheetItem onPress={() => share('facebook')}>
          Share on Facebook
        </ActionSheetItem>
        <ActionSheetItem onPress={() => copyLink()}>
          Copy Link
        </ActionSheetItem>
        <ActionSheetSeparator />
        <ActionSheetItem destructive onPress={() => report()}>
          Report Post
        </ActionSheetItem>
        <ActionSheetCancel>Cancel</ActionSheetCancel>
      </ActionSheetContent>
    </ActionSheet>
  )
}
```

## Controlled

```tsx
import { useState } from 'react'

function ControlledActionSheet() {
  const [open, setOpen] = useState(false)

  return (
    <>
      <Button onPress={() => setOpen(true)}>Show Sheet</Button>
      <ActionSheet open={open} onOpenChange={setOpen}>
        <ActionSheetContent>
          <ActionSheetItem onPress={() => console.log('Selected')}>
            Option
          </ActionSheetItem>
          <ActionSheetCancel>Cancel</ActionSheetCancel>
        </ActionSheetContent>
      </ActionSheet>
    </>
  )
}
```

## Props

### ActionSheet

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |

### ActionSheetTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Trigger element |
| `asChild` | `boolean` | `false` | Render as child element |
| `style` | `ViewStyle` | - | Additional styles |

### ActionSheetContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content |
| `showHandle` | `boolean` | `true` | Show drag indicator |
| `style` | `ViewStyle` | - | Additional styles |

### ActionSheetHeader

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Header content |
| `style` | `ViewStyle` | - | Additional styles |

### ActionSheetTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Title text |
| `style` | `TextStyle` | - | Additional styles |

### ActionSheetDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Description text |
| `style` | `TextStyle` | - | Additional styles |

### ActionSheetItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Item content |
| `onPress` | `() => void` | - | Press handler |
| `disabled` | `boolean` | `false` | Disable the item |
| `destructive` | `boolean` | `false` | Destructive styling (red) |
| `icon` | `ReactNode` | - | Icon element |
| `style` | `ViewStyle` | - | Additional styles |
| `textStyle` | `TextStyle` | - | Additional text styles |

### ActionSheetCancel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Button content |
| `onPress` | `() => void` | - | Press handler (called before close) |
| `style` | `ViewStyle` | - | Additional styles |
| `textStyle` | `TextStyle` | - | Additional text styles |

### ActionSheetSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

## Features

- Slides up from bottom with spring animation
- Tap backdrop to close
- Swipe down to dismiss (gesture)
- Cancel button always at bottom, visually separated
- iOS-style appearance (rounded top corners, grouped items)
- Full-width action items
