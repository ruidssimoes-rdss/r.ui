# Textarea

A multi-line text input component with auto-grow support.

## Installation

```bash
npx r-ui add textarea
```

## Usage

```tsx
import { Textarea } from '@r-ui/react-native'

export default function MyComponent() {
  const [value, setValue] = useState('')

  return (
    <Textarea
      label="Message"
      placeholder="Type your message..."
      value={value}
      onChangeText={setValue}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Textarea label text |
| `placeholder` | `string` | - | Placeholder text |
| `value` | `string` | - | Current value |
| `onChangeText` | `(text: string) => void` | - | Change handler |
| `disabled` | `boolean` | `false` | Disable the textarea |
| `error` | `boolean` | `false` | Show error state |
| `helperText` | `string` | - | Helper or error text |
| `rows` | `number` | `4` | Number of visible rows |
| `maxLength` | `number` | - | Maximum character count |
| `showCount` | `boolean` | `false` | Show character count |
| `autoGrow` | `boolean` | `false` | Auto-grow height based on content |
| `maxRows` | `number` | `10` | Maximum rows when auto-growing |
| `style` | `ViewStyle` | - | Container styles |
| `inputStyle` | `TextStyle` | - | Textarea text styles |
