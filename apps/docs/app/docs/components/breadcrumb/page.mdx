# Breadcrumb

Display navigation path with customizable separators.

## Installation

```bash
npx r-ui add breadcrumb
```

## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
} from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Breadcrumb>
      <BreadcrumbList>
        <BreadcrumbItem>
          <BreadcrumbLink onPress={() => navigate('/')}>Home</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbItem>
          <BreadcrumbLink onPress={() => navigate('/docs')}>Docs</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbItem>
          <BreadcrumbPage>Components</BreadcrumbPage>
        </BreadcrumbItem>
      </BreadcrumbList>
    </Breadcrumb>
  )
}
```

## Props

### Breadcrumb

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `separator` | `ReactNode` | Chevron icon | Custom separator element |
| `style` | `ViewStyle` | - | Additional container styles |

### BreadcrumbList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### BreadcrumbLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onPress` | `() => void` | - | Press handler for navigation |
| `style` | `ViewStyle` | - | Additional styles |
| `textStyle` | `TextStyle` | - | Additional text styles |

### BreadcrumbSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | Chevron icon | Custom separator content |
| `style` | `ViewStyle` | - | Additional styles |

### BreadcrumbPage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |
| `textStyle` | `TextStyle` | - | Additional text styles |
