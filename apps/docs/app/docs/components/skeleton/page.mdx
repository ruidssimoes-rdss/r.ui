# Skeleton

Loading placeholder with shimmer animation for content that is still loading.

## Installation

```bash
npx r-ui add skeleton
```

## Usage

```tsx
import { Skeleton } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Skeleton width={200} height={20} />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `DimensionValue` | `'100%'` | Width of the skeleton |
| `height` | `DimensionValue` | - | Height of the skeleton |
| `variant` | `'text' \| 'circular' \| 'rectangular'` | `'text'` | Shape variant |
| `animate` | `boolean` | `true` | Enable shimmer animation |
| `style` | `ViewStyle` | - | Additional styles |
