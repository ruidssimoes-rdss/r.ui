# Spinner

Loading indicator for asynchronous operations.

## Installation

```bash
npx r-ui add spinner
```

## Usage

```tsx
import { Spinner } from '@r-ui/react-native'

export default function MyComponent() {
  return <Spinner />
}
```

## Loading Button

Combine with Button for loading states.

```tsx
import { Button, Spinner } from '@r-ui/react-native'

function LoadingButton() {
  const [loading, setLoading] = useState(false)

  return (
    <Button disabled={loading} onPress={() => setLoading(true)}>
      {loading ? <Spinner size="sm" color="#0a0a0a" /> : 'Submit'}
    </Button>
  )
}
```

## Full Page Loading

```tsx
import { Spinner } from '@r-ui/react-native'
import { View, StyleSheet } from 'react-native'

function LoadingScreen() {
  return (
    <View style={styles.container}>
      <Spinner size="lg" />
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
})
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spinner size |
| `color` | `string` | `colors.text.primary` | Spinner color |
| `variant` | `'default' \| 'dots'` | `'default'` | Spinner style variant |
| `style` | `ViewStyle` | - | Additional styles |
