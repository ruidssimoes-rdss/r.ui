# Link

Styled navigation links with external link support for mobile.

## Installation

```bash
npx r-ui add link
```

## Usage

```tsx
import { Link } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Link href="https://example.com" external>
      Visit Example
    </Link>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | Link destination URL |
| `external` | `boolean` | `false` | Open link in device browser |
| `underline` | `'always' \| 'hover' \| 'none'` | `'always'` | Underline behavior |
| `variant` | `'default' \| 'muted' \| 'destructive'` | `'default'` | Visual variant |
| `onPress` | `() => void` | - | Custom press handler (overrides href) |
| `disabled` | `boolean` | `false` | Disable the link |
| `style` | `ViewStyle` | - | Additional container styles |
| `textStyle` | `TextStyle` | - | Additional text styles |
