# Table

Data table component with horizontal scrolling for mobile responsiveness.

## Installation

```bash
npx r-ui add table
```

## Usage

```tsx
import { Table, TableHeader, TableBody, TableRow, TableHead, TableCell } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Table>
      <TableHeader>
        <TableRow>
          <TableHead>Name</TableHead>
          <TableHead>Email</TableHead>
          <TableHead>Status</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell>John Doe</TableCell>
          <TableCell>john@example.com</TableCell>
          <TableCell>Active</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  )
}
```

## Props

### Table

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `T[]` | - | Array of data objects |
| `columns` | `Column<T>[]` | - | Column definitions |
| `stickyHeader` | `boolean` | `false` | Keep header visible during scroll |
| `style` | `ViewStyle` | - | Additional styles |

### Column

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `key` | `string` | Required | Data key to display |
| `header` | `string` | Required | Column header text |
| `width` | `number` | - | Fixed column width |
| `align` | `'left' \| 'center' \| 'right'` | `'left'` | Text alignment |
| `render` | `(value, row, index) => ReactNode` | - | Custom cell renderer |

### TableHead / TableCell

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'left' \| 'center' \| 'right'` | `'left'` | Text alignment |
| `width` | `number` | - | Fixed width |
| `style` | `ViewStyle` | - | Additional styles |

### TableRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `selected` | `boolean` | `false` | Highlight row as selected |
| `style` | `ViewStyle` | - | Additional styles |
