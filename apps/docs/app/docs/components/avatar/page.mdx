# Avatar

User profile image with fallback support for displaying initials.

## Installation

```bash
npx r-ui add avatar
```

## Usage

```tsx
import { Avatar } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Avatar
      src="https://example.com/avatar.jpg"
      alt="User avatar"
      fallback="JD"
    />
  )
}
```

## Composable API

For more control, use the sub-components directly.

```tsx
import { Avatar, AvatarImage, AvatarFallback } from '@r-ui/react-native'

<Avatar size="lg">
  <AvatarImage src="https://example.com/avatar.jpg" alt="User" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```

## Props

### Avatar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Alt text for accessibility |
| `fallback` | `string` | - | Fallback text (usually initials) |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Avatar size |
| `style` | `ViewStyle` | - | Additional container styles |

### AvatarImage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Alt text for accessibility |
| `style` | `ImageStyle` | - | Additional image styles |

### AvatarFallback

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Fallback content |
| `style` | `TextStyle` | - | Additional text styles |
