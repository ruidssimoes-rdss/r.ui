## Installation

```bash
npx r-ui add time-picker
```

## Usage

```tsx
import {
  TimePicker,
  TimePickerTrigger,
  TimePickerContent,
  TimePickerWheel,
  TimeValue,
} from '@r-ui/react-native'

export default function MyComponent() {
  const [time, setTime] = useState<TimeValue | null>(null)

  return (
    <TimePicker value={time} onValueChange={setTime}>
      <TimePickerTrigger placeholder="Select time..." />
      <TimePickerContent>
        <TimePickerWheel />
      </TimePickerContent>
    </TimePicker>
  )
}
```

## Features

- **Hour/minute wheels**: Scrollable selection with visual feedback
- **12/24-hour format**: Switch between AM/PM and 24-hour modes
- **Minute intervals**: 1, 5, 15, or 30-minute steps
- **Time constraints**: Set min/max time boundaries
- **Custom format**: Display time in any format
- **Accessible**: Full ARIA support

## Props

### TimePicker

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `TimeValue \| null` | - | Selected time (controlled) |
| `onValueChange` | `(time: TimeValue \| null) => void` | - | Called when time changes |
| `defaultValue` | `TimeValue \| null` | `null` | Default time (uncontrolled) |
| `disabled` | `boolean` | `false` | Disable the picker |
| `placeholder` | `string` | `'Select time...'` | Placeholder text |
| `format` | `string` | `'h:mm A'` or `'HH:mm'` | Time display format |
| `use24Hour` | `boolean` | `false` | Use 24-hour format |
| `minuteInterval` | `number` | `1` | Minute step (1, 5, 15, 30) |
| `minTime` | `TimeValue` | - | Minimum selectable time |
| `maxTime` | `TimeValue` | - | Maximum selectable time |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Called when open state changes |

### TimePickerTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Override placeholder text |
| `style` | `ViewStyle` | - | Additional styles |

### TimeValue Type

```tsx
interface TimeValue {
  hours: number   // 0-23
  minutes: number // 0-59
}
```

### Format Tokens

| Token | Example | Description |
|-------|---------|-------------|
| `HH` | 14 | Hour (24-hour, 2 digits) |
| `H` | 14 | Hour (24-hour, 1-2 digits) |
| `hh` | 02 | Hour (12-hour, 2 digits) |
| `h` | 2 | Hour (12-hour, 1-2 digits) |
| `mm` | 05 | Minutes (2 digits) |
| `m` | 5 | Minutes (1-2 digits) |
| `A` | PM | AM/PM uppercase |
| `a` | pm | AM/PM lowercase |
