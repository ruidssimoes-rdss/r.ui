# Tooltip

Hint text that appears on hover (web) or long-press (mobile).

## Installation

```bash
npx r-ui add tooltip
```

## Usage

```tsx
import { Tooltip, TooltipTrigger, TooltipContent } from '@r-ui/react-native'
import { Button } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Tooltip content="This is a helpful hint">
      <TooltipTrigger>
        <Button>Hover me</Button>
      </TooltipTrigger>
      <TooltipContent />
    </Tooltip>
  )
}
```

## With Delay

Control how long the user must hover/press before the tooltip appears.

```tsx
// Quick tooltip (100ms)
<Tooltip content="Quick hint" delay={100}>
  <TooltipTrigger>
    <Button>Quick</Button>
  </TooltipTrigger>
  <TooltipContent />
</Tooltip>

// Slow tooltip (1 second)
<Tooltip content="Slow hint" delay={1000}>
  <TooltipTrigger>
    <Button>Slow</Button>
  </TooltipTrigger>
  <TooltipContent />
</Tooltip>
```

## Platform Behavior

- **Mobile (iOS/Android)**: Tooltip appears on long-press and disappears when the press ends
- **Web**: Tooltip appears on hover after the delay and disappears when the mouse leaves

```tsx
// The same code works on both platforms
<Tooltip content="Platform-aware tooltip">
  <TooltipTrigger>
    <Button>Interactive</Button>
  </TooltipTrigger>
  <TooltipContent />
</Tooltip>
```

## Custom Styling

```tsx
<Tooltip content="Custom styled tooltip">
  <TooltipTrigger>
    <Button>Styled</Button>
  </TooltipTrigger>
  <TooltipContent
    style={{ backgroundColor: '#3b82f6' }}
    textStyle={{ color: '#ffffff' }}
  />
</Tooltip>
```

## Props

### Tooltip

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `string` | - | Tooltip text content |
| `side` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` | Side to show tooltip |
| `delay` | `number` | `300` | Delay before showing (ms) |

### TooltipTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactElement` | - | Trigger element |
| `style` | `ViewStyle` | - | Additional styles |

### TooltipContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional container styles |
| `textStyle` | `TextStyle` | - | Additional text styles |
