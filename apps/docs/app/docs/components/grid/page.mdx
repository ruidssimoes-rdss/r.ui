# Grid

Grid layout system for React Native with flexible column spanning.

## Installation

```bash
npx r-ui add grid
```

## Usage

```tsx
import { Grid, GridItem } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Grid columns={3} gap={4}>
      <GridItem>Item 1</GridItem>
      <GridItem>Item 2</GridItem>
      <GridItem>Item 3</GridItem>
      <GridItem colSpan={2}>Wide Item</GridItem>
      <GridItem>Item 5</GridItem>
    </Grid>
  )
}
```

## Props

### Grid

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 12` | `2` | Number of columns |
| `gap` | `0 \| 1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 8 \| 10 \| 12` | `4` | Gap between all items |
| `rowGap` | `0 \| 1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 8 \| 10 \| 12` | - | Gap between rows (overrides gap) |
| `columnGap` | `0 \| 1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 8 \| 10 \| 12` | - | Gap between columns (overrides gap) |
| `style` | `ViewStyle` | - | Additional styles |

### GridItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `colSpan` | `number` | `1` | Number of columns to span |
| `style` | `ViewStyle` | - | Additional styles |
