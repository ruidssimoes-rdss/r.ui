# Card

Container component with elevation or border styling for grouping related content.

## Installation

```bash
npx r-ui add card
```

## Usage

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        <Text>Card content</Text>
      </CardContent>
      <CardFooter>
        <Button>Action</Button>
      </CardFooter>
    </Card>
  )
}
```

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'elevated' \| 'outlined' \| 'ghost'` | `'elevated'` | Visual style variant |
| `padding` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Internal padding |
| `style` | `ViewStyle` | - | Additional container styles |

### CardHeader

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### CardContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### CardFooter

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | - | Additional styles |

### CardTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `TextStyle` | - | Additional text styles |

### CardDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `TextStyle` | - | Additional text styles |
