# Carousel

Horizontal scrolling carousel with dot indicators and navigation arrows.

## Installation

```bash
npx r-ui add carousel
```

## Usage

```tsx
import { Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext, CarouselDots } from '@r-ui/react-native'

export default function MyComponent() {
  return (
    <Carousel>
      <CarouselContent>
        <CarouselItem>
          <View style={{ height: 200, backgroundColor: 'blue' }} />
        </CarouselItem>
        <CarouselItem>
          <View style={{ height: 200, backgroundColor: 'green' }} />
        </CarouselItem>
        <CarouselItem>
          <View style={{ height: 200, backgroundColor: 'purple' }} />
        </CarouselItem>
      </CarouselContent>
      <CarouselPrevious />
      <CarouselNext />
      <CarouselDots />
    </Carousel>
  )
}
```

## Props

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `autoplay` | `boolean` | `false` | Enable auto-advance |
| `interval` | `number` | `5000` | Autoplay interval in ms |
| `loop` | `boolean` | `false` | Loop back to start/end |
| `showDots` | `boolean` | `true` | Show dot indicators |
| `showArrows` | `boolean` | `true` | Show navigation arrows |
| `onSlideChange` | `(index: number) => void` | - | Callback on slide change |
| `style` | `ViewStyle` | - | Additional styles |

### CarouselContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | CarouselItem children |
| `style` | `ViewStyle` | - | Additional styles |

### CarouselItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Slide content |
| `style` | `ViewStyle` | - | Additional styles |

### CarouselPrevious / CarouselNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Custom arrow content |
| `disabled` | `boolean` | - | Force disabled state |
| `style` | `ViewStyle` | - | Additional styles |

### CarouselDots

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `activeColor` | `string` | `colors.text.primary` | Active dot color |
| `inactiveColor` | `string` | `colors.text.muted` | Inactive dot color |
| `style` | `ViewStyle` | - | Additional styles |
